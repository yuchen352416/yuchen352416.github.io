HBase 日常使用命令

HBase 初体验

安装 HBase

docker pull harisekhon/hbase

创建HBase容器

docker run -d --name=hbase -p 2181:2181 -p 8080:8080 -p 8085:8085 -p 9090:9090 -p 9095:9095 -p 16000:16000 -p 16010:16010 -p 16201:16201 -p 16301:16301 harisekhon/hbase

启动 HBase 服务

docker start hbase

停止 HBase 服务

docker stop hbase

启动 HBase 命令行

docker exec -ti hbase /bin/bash

进入 shell

hbase shell

退出 shell

exit

建表

create ‘SCORE’, {NAME => ‘F’, VERSIONS => 1}, {NAME => ‘M’, VERSIONS => 1}

删除表

disable ‘SCORE’
drop ‘SCORE’

新增记录

put '<table>', '<row_key>', '<column_family:column_qualifier>', '<cell_value>', [<timestamp>]

例子

put ‘SCORE’, ‘row_key_1’, ‘M:1’, ‘1’

删除单元格

delete '<table>', '<row_key>', '<column_family:column_qualifier>' [, <timestamp>]

例子

delete ‘SCORE’, ‘row_key_1’, ‘M:1’

删除行

deleteall '<table>', '<row_key>' [, '<column_family:column_qualifier>', <timestamp>]

例子

deleteall ‘SCORE’, ‘row_key_1’

清空表中数据

truncate <table>

例子

truncate ‘SCORE’

扫描表

scan <table>, {COLUMNS => [ <family:column>,.... ], LIMIT => num}

例子

scan ‘SCORE’, {LIMIT=>2}

查询某行记录

get <table>,<rowkey>,[<family:column>,....]

注:

  • table 表
  • row_key 行
  • column_family 列簇
  • column_qualifier 列
