Bootstrap 插件(DataTables, DatePicker)

发表于 2018-07-23 | 分类于 JavaScript

Bootstrap 常用的表格插件(DataTables), 日期插件(DatePicker) 和 下拉列表控件(BootStrap-Select)

表格插件(DataTables)

  1. 引入依赖文件
    • /vendor/datatables/js/jquery.dataTables.min.js
    • /vendor/datatables-plugins/dataTables.bootstrap.min.js
    • /vendor/datatables-responsive/dataTables.responsive.js
    • /vendor/datatables-plugins/dataTables.bootstrap.css
    • /vendor/datatables-responsive/dataTables.responsive.css

  1. 实际使用

  2. 常用配置

    • "order": [[ 1, "desc" ]] 默认第二列降序排列
    • "columnDefs": [{ "orderable": false, "targets": 0 } ] 第一列不进行排序
    • "initComplete" : function() {} 初始化表格时, 执行的方法

  3. 常用方法

    • 自定义检索条件
    • 刷新页面后, 回显页码

    注意: DataTables的配置项中, 要加入 "bDestroy": true

日期插件(DatePicker)

  1. 引入依赖文件

    • /vendor/bootstrap-datepicker/js/bootstrap-datepicker.js
    • /vendor/bootstrap-datepicker/locales/bootstrap-datepicker.zh-CN.min.js
    • /vendor/bootstrap-datepicker/css/bootstrap-datepicker.css

  2. 实际使用

  3. 常用配置

    • format: 'yyyy-mm' 配置日期格式
    • startDate: '-12m' 配置可选择的开始日期
    • endDate: '0d' 配置可选择的结束日期
    • language: 'zh-CN' 配置语言
    • autoclose: true 自动关闭
    • startView: 'months' 开始视图为选择月份
    • maxViewMode:'years' 最高层视图为 选择年份
    • minViewMode:'months' 最底层视图为 选择月份

下拉列表控件(BootStrap-Select)

  1. 引入依赖文件
    • /vendor/bootstrap-select/dist/js/bootstrap-select.min.js
    • /vendor/bootstrap-select/dist/js/i18n/defaults-zh_CN.min.js
    • /vendor/bootstrap-select/dist/css/bootstrap-select.min.css

  1. 实际使用

  2. 常用配置

    • data-live-search="true" 开启搜索
    • multiple 开启复选
    • title="--请选择--" 默认显示内容
