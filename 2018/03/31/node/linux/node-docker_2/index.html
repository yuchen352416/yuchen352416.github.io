Docker 安装 MySQL容器

1. 下载 MySQL镜像

docker pull mysql/mysql-server

2. 启动容器

docker run -p 3306:3306 --name=mysql -d mysql/mysql-server

3. 查看MySQL默认密码

docker logs mysql

4. 登录后修改密码

set password = password('123456');

5. 解决一下远程登录的问题

update user set host = '%' where user = 'root';
