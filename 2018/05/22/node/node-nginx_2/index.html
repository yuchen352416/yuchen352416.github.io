MacOS 安装与配置 Nginx

Nginx 安装

brew install nginx

Nginx 涉及到的路径

  • /usr/local/cellar/nginx/1.13.12 安装目录
  • /usr/local/etc/nginx 配置文件
  • /usr/local/var/log/nginx 日志文件

配置虚拟主机

Nginx 常用命令

  • nginx -t 测试配置是否正确
  • nginx -s reload 重新加载配置文件
  • nginx -s stop 立即停止服务
  • nginx -s quit 优雅的停止服务
  • nginx -s reopen 重新打开日志
