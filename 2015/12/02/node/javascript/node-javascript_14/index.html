父页面获取iframe 子页面的js变量值

方法只有一种：

步骤：

1、获取iframe节点

2、获取iframe中的变量

示例

a.html
<iframe id='x' name='x' src='b.html'/>

b.html

在a.html中获取y，则可以通过以下方式
document.frames['x'].y //先获取frame,在获取y

子页面获取父页面js变量

父页面

<input type="hidden" id="result" name="result" value="" />

子页面

<input type="hidden" id="_bh${da.bh}" value='${da.bh},${da.name},${da.sfzh},${da.lxdh}'/>
